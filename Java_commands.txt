Be in the d-sync directory.

Compile everything:
javac src/*.java

Run from terminal :
java -cp . src.Main

Create exectutable jar file :
jar -cvmf manifest.mf Dsync.jar ./classFiles ./src ./LICENSE.md ./README.md

(The Main-Class attribute is defined in manifest.mf)

Compile and run :
javac src/*.java ; java -cp . src.Main

Compile and create jar :
javac src/*.java ; jar -cvmf manifest.mf Dsync.jar ./classFiles ./src ./LICENSE.md ./README.md
